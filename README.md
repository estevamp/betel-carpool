# Carpool Betel

Aplicacao web para gestao de transporte de betelitas.

## Desenvolvimento local

Requisitos:
- Node.js 20+
- npm

Comandos:

```sh
npm install
npm run dev
```

## Build e preview

```sh
npm run build
npm run preview
```

## Stack

- Vite
- React + TypeScript
- Tailwind CSS
- shadcn/ui
